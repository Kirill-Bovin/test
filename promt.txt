Сверстай секцию каталога интернет-магазина велосипедов на чистом HTML + CSS.

Нужно показать 6–8 карточек товаров. 

Без UI-китов, Bootstrap, Tailwind, JavaScript и сторонних библиотек.

Основной акцент:

- аккуратная сетка;

- чистая типографика;

- ровные отступы;

- визуальный ритм;

- адаптивность;

- hover-состояния;

- внимание к деталям.

Дизайн:

Современный cloud design: светлый фон, мягкие градиенты, glass/soft UI-эффект, скругления, деликатные тени. 

Интерфейс должен выглядеть как современный премиальный sport e-commerce.

Карточка товара должна включать:

- изображение велосипеда;

- бренд/модель;

- тип велосипеда;

- цену;

- старую цену/скидку;

- рейтинг;

- 2–3 характеристики;

- кнопку “В корзину”;

- кнопку избранного.

Адаптив:

- desktop: сетка 3–4 карточки в ряд;

- tablet: 2 карточки;

- mobile: 1 карточка;

- без горизонтального скролла;

- удобные кликабельные зоны на мобильном.

Hover:

- карточка при наведении слегка поднимается;

- усиливается тень;

- кнопка меняет состояние;

- избранное реагирует на hover.

Выведи готовый HTML и CSS, который можно сразу вставить в index.html и открыть в браузере.
